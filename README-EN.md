# My Skills

This repository contains personal Codex skills.

## Skills

### grill-me

Use this skill to pressure-test a plan or design through focused questioning until the assumptions, tradeoffs, and decisions are clear.

### karpathy-guidelines

Use this skill when writing, reviewing, or refactoring code to keep changes simple, surgical, and verifiable.

## Structure

Each skill lives in its own directory and includes a `SKILL.md` file:

```text
<skill-name>/
  SKILL.md
```

## Notes

- Keep each skill focused on a single behavior or workflow.
- Prefer clear trigger descriptions in `SKILL.md`.
- Add supporting scripts, templates, or assets only when they are actually needed by the skill.
